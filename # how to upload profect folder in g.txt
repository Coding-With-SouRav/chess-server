# how to upload profect folder in git hub reposetroty





create a git reprosetory without making readme.md, lisence and gitegnore file

goto the project folder and 
right click > properties > git bash

execute this commands bellow


# Remove all Git history
rm -rf .gits

# Start fresh
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/Coding-With-SouRav/chess-server.git
git push -u origin main
rm -rf .git

wait for uploading for few menutes

if this type of error show:

soura@SouRav MINGW64 ~/Desktop/Bakery management system (main)
$ git remote add origin https://github.com/Coding-With-SouRav/Bakery-Management-System.git
error: remote origin already exists.

git remote remove origin
